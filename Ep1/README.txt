Como gerar o jar executável:
    make

Como compilar o EP:
    make compile

Como rodar o jar executavel;
    java -jar ep1.jar ARG1 ARG2
  onde ARG1 é o número do algoritmo escolhido e 
       ARG2 é o N para tal algoritmo.
  
Para limpar a pasta:
    make clean
