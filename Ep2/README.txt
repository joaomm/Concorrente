Como gerar o jar executável:
    make

Como compilar o EP:
    make compile

Como rodar o jar executavel;
    java -jar ep2.jar ARG
  onde ARG é o nome do arquivo de configuracao de maquinas.
  
Para limpar a pasta:
    make clean
